# 🧙‍♂️ Sistema de Monitoria - POO Básico

![Java](https://img.shields.io/badge/java-%23ED8B00.svg?style=for-the-badge&logo=openjdk&logoColor=white)

Projeto desenvolvido para a disciplina de Programação Orientada a Objetos (POO), com o objetivo de aplicar na prática os conceitos fundamentais da orientação a objetos utilizando Java.

---

## 📌 Sobre o Projeto

Este sistema simula um gerenciamento simples de personagens de RPG via console.  
O usuário pode criar personagens, escolher suas classes e executar ações com eles.

---

## ⚙️ Funcionalidades

- Criar personagens do tipo:
  - 🗡️ Guerreiro
  - 🔮 Mago
- Definir nome do personagem
- Personagens iniciam no nível 1
- Armazenamento em lista (`ArrayList`)
- Listagem de personagens criados
- Execução de ações (ataque)
- Menu interativo via console

---

## 🧠 Conceitos Aplicados

- Encapsulamento
- Herança
- Polimorfismo
- Tratamento de exceções (`try/catch`)
- Estruturas de dados (`ArrayList`)

---

## 🗂️ Estrutura do Projeto
```bash
src/

│ ├── Guerreiro.java
│ └── Mago.java
├── controller/
├── service/
└── Main.java
```

---

## ▶️ Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/KSelyson/Sys-Monitoria.git
```
2. Acesse a pasta do projeto:
```bash
cd Sys-Monitoria
```
3. Compile e execute o projeto em sua IDE (IntelliJ, Eclipse, VS Code) ou via terminal.

---

## 📋 Menu do Sistema

```bash
1 - Criar Guerreiro
2 - Criar Mago
3 - Listar personagens e atacar
0 - Sair
```
## 🎯 Objetivo Acadêmico

Este projeto tem como finalidade consolidar o entendimento dos pilares da Programação Orientada a Objetos através de uma aplicação prática simples e funcional.
